# 第1章 Flask Web 开发基础

# Flask框架介绍

Flask框架依赖两个外部库：Werkzeug 和 Jinjia2.

  • Werkzeug，是一个WSGI工具集，是Web应用程序和多种服务器之间的标准Python接口。
  • Jinja2：负责渲染模板，将由HTML，CSS和JavaScript组成的模板文件显示出来。

# 简单示例

import flask

app = flask.Flask(__name__)
@app.route('/')

if __name__ == '__main__':
    app.run()

在Flask框架的URL中，除了可以使用字符串类型参数外，还可以使用如下3种类型的参数。

  • int: 表示整型的参数，例如 "/hello/1"
  • float: 表示浮点型的参数，例如 "/hello/1.1"
  • path: 表示使用斜杠，例如 "/hello/"

如下面的示例：

from flask import Flask
app = Flask (__name__)

@app.route('/blog/<int:ID>')
def show_blog(ID):
    return '我的年龄是: %d' % ID + '岁!'

@app.route('/rev/<float:No>')
def revision(No):
    return '我身上只有%f' % No + '元钱了!'
    
if __name__ == '__main__':
    app.run()

# 网页重定向

Flask Web程序中处理URL请求的时候，可以使用网页重定向方法url_for()来到指定的URL。方法url_for()的语法格式如下。

url_for(endpoint, **values)
1

url_for()方法可以传递如下两个参数：

  • endpoint: 表示将要传递的函数名。
  • **values : 是个关键字参数，即有多个 key = value的形式参数。

下面的示例中，如果访问的是/user/admin 的路径，那么就直接重定向到 /admin

from flask import Flask, redirect, url_for
app = Flask(__name__)
@app.route('/admin')
def hello_admin():
    return '你好管理员'
@app.route('/guest/<guest>')
def hello_guest(guest):
    return '你好%s, 你是游客!' % guest
@app.route('/user/<name>')
def hello_user(name):
    if name == 'admin':
        return redirect(url_for('hello_admin'))
    else:
        return redirect(url_for('hello_guest', guest = name))

if __name__ == '__main__':
    app.run(debug = True)

# 第2章 使用Flask模板

# 什么是模板，为什么需要模板呢

正常在一个web程序中，通过业务逻辑函数得到数据后，需要有一个页面来展示显示，也就是有个HTTP的响应，对于Web程序来说，HTTP响应一般就是一个HTML文件。

在Flask Web开发中，一般的做法就是提供一个HTML模板文件，然后将数据传入到模板中，在渲染HTML模板文件后会得到最终需要的HTML响应文件。

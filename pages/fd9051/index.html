英语复试&综合复试

# 复试英语简历梳理

复试所占比重： 40% - 60%

复试英语的考察范围，(笔试)+面试，其中面试是一定会考的

  • 官网搜索 xx大学研究生院
  • 查找当年或前一年的招生简章 + 招生专业目录
  • 19年的招生简章，是18年的9月份发布的，基本上就是研究生初试的报名的前后，有些会出现在网报公告里面
  • 给学校研招办打电话、发邮件；给学校的教学秘书、系主任发邮件
  • 问目标院校目标专业的师哥师姐
  • 英文问你，你就英文回答；中文问你，你就中文回答

# 英语面试题型介绍

  1. 英文简历
  2. 自我介绍，一般是1~2分钟，最多不超过3分钟
  3. 老师提问(日常话题、专业话题、热点话题)
  4. 听/读 材料之后的问答或翻译
  5. 小组讨论

# 回答策略

  4. 最后，再来做一下总结

回答7句话就可以了，最少5句话，最多10句话就可以了。

# 中文简历梳理

找一些专业对应行业的热点事件

学校找不到，到学院网页上面来找一找

验证他，通过他近期发表的期刊，论文来研究。

通过 "中国教育在线"来检索

了解研究生期间都学些什么，通过研究生官网

多做一些基础工作，对于选择和报考有利。

研究生一年级的教材，做更高维度的准备，去面对一个中等维度的考试。

会根据简历上面的问题来提问，来问些衍生的问题。这就给了我们机会，能把自己梳理出来的优势给梳理出来，把简历当中，想让老师提问的问题给梳理出来，这样老师来问，自己也就准备过了。

在学术水平、能力上比我们高，但是在人格上是一致的。

让导师在短短的20分钟内，了解自己，让老师感觉到这个学生，自己能够带出来。

自信一点，每个人都有自己的优势；

要从真实的经历中，提炼自己的亮点。

让导师看到你对自己负责任的态度，比如说简历中不能出现错别字，照片照一张好的，简历排版稍微漂亮一些；通过自己的简历，让导师能够感受到，你对专业/学校的兴趣和重视；让导师感受到 录取你是彼此的双赢。

简历写的每一行下去，想一想老师会问哪些问题，这句话有没有给自己挖坑；简历写完了之后，旁边应该有一页A4纸，罗列出来6-8个问题；

有些同学本科不好，初始成绩不好，四六级成绩不好，不好的内容，我们尽量避开。

宗旨是，不说假话，真话不全说。

# 简历模块梳理

模块1-3，有啥说啥，个人邮箱、电话要写对，个人照片要是证件照。

初试的情况，分数高就写，分数低就不要写了。分数不是优势的时候，就不要在这写。

模块3，如果是主专业的，这一项就写上。挑我们的优势去说

模块4-6，切记平铺直叙，利用关键事件梳理方法(STAR)，

简历先做加法，然后再来做减法，例如可以先写3页，然后浓缩到1页。每写一行，都要设想一下，这个可能会怎么来问。

能用数据的用数据，能简洁的要简洁。

考的不好的话，在描述兴趣爱好的时候，要突出这个兴趣爱好是有一定的价值的，要么就是和专业有一定的相关的。

有工作经验的，要凸显工作能力。

如果要写自我评价，一定要在自我评价中，要分层次，可以从学习上、从生活上、从兴趣爱好上，从未来规划上，去进行自我评价。甚至在自我评价的时候，可以主动回答一些问题，例如为什么要跨专业。

这10个模块，没有必要所有都写上，写3-4个就够了。主要还是要抓住自己的优势来写，简历最好就是1页纸。先做加法，再做减法

# 简历样板

如果越到后面越严肃了，如果遇到 问老师的问题，要问专业问题，不要问啥问题。

"老师，我想暑假期间，提前了解咋们专业"，我们要先入为主，要假设自己已经被录取了，"老师，我想提前看一些专业上的书籍，你建议我先来看哪一本，您建议我先看哪些"

给了一个压力测试型问题，要接住这个问题，"老师，上面您问的哪个问题，下来您建议我看哪些资料和期刊呢?"

每个简历下面，都要给自己6-8个问题，

把自己称述写长一些，写详细一些。

我是从哪些角度，获取到了这个好的信息， 结合我自身的情况，我觉得目前这个专业，依旧是很适合我的。

第二，我有能力有信心学好的。

有些问题，不要直接回答，稍微绕一绕，

# 问题4

确实对这个专业了解的有些偏颇，但是我觉得 有的时候，偏执就是一种热爱。

我了解的资料优先，招生简章，曾经找过学长学姐咨询，给了一些中肯的建议；问了学校的辅导老师；也从网上查了一些资料，综合来看，我觉得这个专业是非常适合我的。

可能了解的渠道有限，对于这个专业的了解有些偏执，但是我始终认为，我还是适合这个专业的。

# 问题5

问的就是有没有能力，有没有信心学下来。

我在准备考研的时候，就听学长学姐说过这个问题，这个问题不仅出现在跨专业的同学身上，本专业的也有。

第一，我会补齐我的基础，尽可能的跟上课程，第一时间找老师解决问题，以目前的学习能力和学习状态，我相信劝退这件事，不会发生在我的身上。

# 问题6

第1，要具备专业课很强的学习能力。读研期间，锻炼自己很强的文献论文的阅读能力。

第2，读大量文献的同时，要结合论文和文献中的问题，思考我们这个专业未来可以创新的地方。

第3，还是要有很强的团队合作能力，

第4，还要有很强的抗压能力。作为跨专业的学生，在基础知识上可能会比其他同学差一些，既要补齐基础，又要有相应专业学习，这样对压力还是很大的。很多实验都是一次次的做，我们也要有足够的耐心和专业的热爱，才能坚持下去。

第5， 英语能力要好，很多文献和论文都是英文写成的，要有很强的英文能力，才能接触到最新鲜的知识。

# 问题7

这个考察的就是现场的抗压能力。

该转换下概念，就转换下概念。

本科的学的某某专业，觉得不是劣势，反而是我的优势。要有信心

第1，我能够进入复试，说明我的学习能力是符合我们学校的基础要求的。

第2，跨专业的学习，相对于本专业的同学来讲，我了解的会更多，知识面会更广了， 在有些方面，我可能会比其他同学了解的更多。我相信我可以用原本的知识，给我的研究生学习，在创新上带来更多的可能性。

可能我的知识确实是有些欠缺的，但是各位老师放心，我会在暑假期间，补齐自己的短板，好好补习基础。

# 跨专业复试三条建议

# 面试现场基本素质及复试礼仪

在疫情可控的情况下，陆续会回来。

重点关注目标院校的复试要求 什么时候会出，以及复试的细节。

甚至会出现有些学校，有些学院要求线下，有些学院要求是线上，看不同的学院要求不一样。

今年至少会有30% 的学校会采取 线下复试。

考官的提问会越来越灵活，越来越针对性越强

无论是线上还是线下，提前30分钟到场候考，不要迟到。

会提前1-2天，进行设备测试。机位怎么摆

# 复试礼仪三部曲

# 你的样子

干干净净最重要。在穿着上，就和平常是一样就可以了。

表情要自信，从容自定， 要学会和老师 用眼光去交流，不用楞盯着。

谁提问，就可以盯着他来看，自信，不用去怕

你已经走到复试考场里面，就已经证明了你的能力，啥也不用怕。

# 招牌动作

就是站有站姿，坐有坐姿，不要有小动作。

手就放在桌子上就可以了。

# 真实表达

无论是男女，音量要足够高。

声音洪亮是自信的表现，语速不要过快，首先如果语速太快，未必能听得清，其次如果回答太快了，有可能会问更多的问题。

高低起伏，抑扬顿挫。

不要有些不文明的语气词，不要说嗨了。

不要声音小，没有自信，思维混乱，言语重复

对着镜子，或者对着视频，打开手机的前置摄像头 ，多录几遍，自我介绍啊，问的问题，这类的，找找感觉。录完之后，自己再来回放一下，看一看，自己的问题在哪里。

去年就有学生，自己把摄像头打开 对着所有的问题，模拟了2-3轮，然后再来上考场，这是不一样的感觉。

正常来写作业是一个逻辑，写完之后正常来表达，是另外一个逻辑。有准备和没有准备，有练习 和 没有练习 ，这是完全不一样的。

大家要尽可能的多去练习，并且尽可能的刻意去练习。

# 听规则

听规则，不要擅自行动，如果候场的时候 手机要收起来，就收起来。

不害怕、不紧张、不焦躁，多练习。

# 笔试

结合你初试考的专业课，加上人家复试让你看的专业课，认认真真去把这个书看透，还是非常有帮助的。

一本本的看，认认真真的复试，看进去了， 才能考的上。

梳理初试的重点，+ 梳理个人的优势， + 去了解行业的前言问题。梳理你对这个专业的理解。

问啥答啥，优势就是梳理好的那么几点。梳理好统一的答题思路，问到一个大问题，要学会拆解成几个小问题，从不同层次、不同角度去回答这个问题。

学校老师既然知道你是调剂的，不会在你没有学习过的专业课的内容上，去难为你。他要验证你的，是你的学习能力。往年有些调剂的考生也是有笔试的，老师也会忍者给你一个及格的分数，让你进入到面试当中去，和你聊一聊。不要让这个卷子空着，该写要写上。既然是调剂的，老师既然想要你，就不会太难为你的

# 专业课面试

如果问道实践性问题，态度要对，要认怂，实话实说，多说一些未来规划。老师也会知道 我们的实践水平也不会很高。

了解行业的热点和行业的前沿，看老师的论文，看老师的研究方向，看老师可能的研究方向。从谷歌学术、百度学术上来整理我们的问题。导师论文的关键字，导师论文的摘要，每天哪怕做一件，也不会很慌。

专业课热点问题，只能是多收集，多关注，老师的信息这些多去查一查。

我们之所以准备这么多的问题，准备这么多的自我介绍，简历，就是为了引导进入到自己擅长的方面。当我们听到问题的时候，要学会转换，要去灵活应用。说一些自我的规划，甚至可以前置的去回答一些问题。

面对专业课问题，很恐慌的最大的问题就是准备不足。

如果问到完全陌生的问题，又是专业课问题，实在没有办法，就申请换题。申请换题，老师是可以同意的。可以稍微委婉一些，“老师，关于你问到的xxx问题，我还是第一次听说(了解不太多)，能换一个吗？” 申请换题，不会被扣分。

忘了专业，老师这么问，不是不要你，只是好奇，为什么这么查，要诚实说，实话实说，要让老师能够相信的。 整个期间比较迷茫，没有认认真真的学。

在均衡发展的前提下，我会把70%的精力用在科研上，毕竟要先学才能实践，实践是检验真理的唯一标准。

多练习，才能做到不怯场。

梳理优势，梳理回答思路。
